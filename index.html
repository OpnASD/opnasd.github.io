OpenASD

Open source operating system

Very cool and very strong OS, made for cool guys

features:

  • Microkernel - all errors are moved to the user space, so you won't see Kernel Panic
  • It's not Unix - that's what makes this project completely unique
  • libasd - is a small and fast libc, making the OS itself very fast
