What is OpenASD?

OpenASD is an independent operating system. It is a microkernel-based OS, meaning it is simple to implement (less code = fewer bugs) and reliable. It has a fast libc and supports mach-o binaries.

does it have FastFetch?

Yes! You can use Fastfetch, because it's built into the OS at the kernel level (just kidding).
